Get_Record
Retrieves a single dimensional, key-value pair array from the database.
Description
Get_Record(string $sql_query) : array
Parameters
sql_query
Sql Query of type String that fetches data from a database.
Return Value
This function returns a key-value pair array of data corresponding the Sql Query submitted. If no data matches the submitted Sql Query, i.e. The database returns an empty array, then NULL is returned instead of an empty array.
Example
Given the following table - Table_1:
field_id field_1 field_2 field_3
1 Text example one. 100 Apple
2 Text example two. 101 Orange
3 Text example three. 256 Pear
